# Test Task: Patient Registration Backend

## Requirements

Develop an asynchronous backend for a patient registration system.

### Tech stack:
- FastAPI (async)
- SQLAlchemy ORM (PostgreSQL)
- Dependency Injection
- Pytest tests
- Docker Compose demo

## Priorities:
- Working prototype
- Clean and understandable architecture
- Code readability
- Basic security considerations

## Notes:
The solution should be simple but production-minded.
Overengineering should be avoided.
